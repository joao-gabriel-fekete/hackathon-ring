# React + TypeScript + Vite

## Setup
### Dependencies

* You need to have installed NPM or Yarn.

Clone the repository:
```
git clone https://github.com/joao-gabriel-fekete/hackathon-ring.git
```

Inside the directory run Yarn to install dependencies:
```
yarn install
```

Run the environment:
```
npm run dev
```
